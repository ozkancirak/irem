# Irem için küçük sürpriz 

Mobil uyumlu, kalpli ve interaktif tek sayfalik website.

## Çalistirma
- index.html dosyasini çift tiklayip aç.
- Ya da bir statik sunucuda (GitHub Pages vb.) barindir.

## Özellikler
- Mobil-first responsive tasarim
- Tiklayinca uçan kalpler ve animasyonlar
- Düzenlenebilir sevgi notu (dialog)
- Erisilebilirlik için semantik HTML ve etiketler

## Özellestirme
- index.html baslik ve metinleri düzenle
- styles.css içindeki renk degiskenlerini (:root) degistir
- script.js içinde baslangiç kalp sayisini ayarla

Iyi sanslar!
